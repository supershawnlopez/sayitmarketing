### The free stack + monthly workflow for every SEO client
*Last updated: April 2026*

## The Free Stack (Use This for Every Client)

| Tool | Cost | What It Does |
|---|---|---|
| Google Business Profile Insights | Free | Calls, map views, website clicks, search queries |
| Google Search Console | Free | What keywords bring traffic to the site, click-through rates |
| Whitespark Local Rank Tracker | Free tier | Tracks local map pack keyword positions |
| Google Keyword Planner | Free (needs Ads account) | Find local search volume for target keywords |
| Claude / ChatGPT | Already paying | Write GBP posts, review responses, report summaries |

**That's it. No paid SEO software needed at the $199/mo tier.**

## Initial Setup — Do This Once Per Client

### 1. Google Business Profile Access
- Ask client to add you as a **Manager** (not Owner) in their GBP dashboard
- Go to: business.google.com → Manage Profile → Business Profile Settings → Managers → Add
- Needs their Google account email — they approve the invite

### 2. Google Search Console
- Go to: search.google.com/search-console
- Add property → Domain type → enter their domain
- Verify via DNS record (add TXT record in Namecheap DNS)
- Takes 24–48 hrs to populate data
- **Note:** New sites will have very little data for 60–90 days — set expectations upfront

### 3. Whitespark Local Rank Tracker (Free Tier)
- Sign up at: whitespark.ca
- Free tier allows tracking up to 5 keywords per location
- Add their business name + location
- Add 5 target keywords (e.g. "dog groomer Glendale AZ", "pet grooming near me")
- Set tracking to weekly
- Screenshot results monthly for the report

### 4. Directory Listings (One-Time — Bill as Setup)
Submit the business to these manually — takes about 2 hours total:
- Google Business Profile (should already exist)
- Yelp — yelp.com/biz/claim
- Apple Maps — mapsconnect.apple.com
- Bing Places — bingplaces.com
- Yelp (already listed above)
- Yellow Pages — yp.com
- Nextdoor — business.nextdoor.com
- Facebook Business Page (if they don't have one)

**Consistency rule:** Business name, address, phone number must be IDENTICAL across every listing. Even small differences (St. vs Street) hurt rankings.

## Monthly Workflow — 2 to 3 Hours Per Client

### Week 1 — Google Post #1
- Write a post using Claude: service highlight, seasonal offer, or tip
- Post directly in GBP dashboard → Add Update
- Include a photo if possible (client sends, or use a relevant stock image)
- Add a call-to-action button (Call Now or Book)

**Prompt for Claude:**
> Write a Google Business Profile post for [Business Name], a [service type] in [city]. Topic: [seasonal/service/promo]. Keep it under 150 words, friendly tone, end with a call to action to call or book.

### Week 2 — Google Post #2 + Review Check
- Write and post second GBP update
- Check for new reviews in GBP dashboard
- Respond to any new reviews (positive AND negative) — use Claude to draft responses
- Send client their review request reminder text/script if they haven't been asking

### Week 3 — Google Post #3 + Photo Update
- Write and post third GBP update
- Upload 2–3 new photos to GBP (ask client to send fresh photos monthly)
- Check profile for anything outdated (hours, services, description)

### Week 4 — Google Post #4 + Pull Report Data
- Write and post fourth GBP update
- Pull GBP Insights screenshot (calls, views, searches)
- Pull Search Console data (top queries, impressions, clicks)
- Pull Whitespark ranking snapshot
- Compile into monthly report (see template below)

## Review Request System — Give This to Every Client

Train the client to send this text after every appointment:

**Text template (client sends to customer):**
> "Hey [name]! Thanks so much for coming in today 🐾 If you have a sec, would you mind leaving us a quick Google review? It helps so much — here's the link: [Google Review Link]"

**How to get their Google Review Link:**
1. Go to their GBP dashboard
2. Click "Ask for reviews"
3. Copy the short link Google generates
4. Give it to the client — they paste it into every follow-up text

## What to Tell the Client About Timing

Set these expectations at signup — every time, no exceptions:

| Timeline | What Happens |
|---|---|
| Days 1–14 | Profile optimized, listings submitted, posts begin |
| Days 15–45 | Profile looks active, reviews start building, baseline rankings captured |
| Days 45–90 | First ranking movements visible, more calls/clicks in GBP insights |
| Month 4+ | Compounding effect — rankings improve as Google sees consistent activity |

**Key line to use:** *"Google rewards consistency. The more we show up every month, the higher you rank. Think of it like a gym membership — results come from showing up regularly."*

## Upsell Triggers — Watch for These

| Signal | Upsell |
|---|---|
| Ranking well but low conversions | Website CRO review |
| Opens soon / needs traffic now | Add Google Ads campaign |
| Reviews growing, GBP solid | Upgrade to Dominator ($299) for service area pages |
