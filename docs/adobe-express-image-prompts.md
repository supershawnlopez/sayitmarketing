# Adobe Express Image Prompts (Website)

Use these exact file names so no code changes are needed.

## 1) Homepage hero image
- File name: `hero-visual.svg` (or `hero-visual.png`)
- Save to: `assets/hero-visual.svg`
- Size: `1600 x 1200`
- Prompt:
`Create a modern website mockup scene for a digital services company. Show a clean desktop and mobile screen with a booking form, CTA button, and analytics card. Style: premium, minimal, Apple-inspired, soft daylight, white and blue accents, no logos, no readable brand text, realistic but clean, high contrast, professional.`

## 2) Print services feature image
- File name: `section-visual.svg` (or `section-visual.png`)
- Save to: `assets/section-visual.svg`
- Size: `1600 x 900`
- Prompt:
`Create a high-end product layout showing business cards, flyers, menu sheet, storefront sign mockup, and a shipping box label. The theme should communicate white-label print fulfillment and nationwide drop shipping. Clean studio lighting, white background, subtle blue accents, realistic materials, no logos, no readable text.`

## 3) Quote page social image
- File name: `hero.svg` (or `hero.png`)
- Save to: `assets/hero.svg`
- Size: `1200 x 630`
- Prompt:
`Create a conversion-focused landing page visual with a short quote request form, message bubbles, and a scheduling card. Modern UI, clean whitespace, white and blue palette, minimal shadows, premium and trustworthy look, no logos, no readable text.`

## Optional 4) Nationwide service collage
- File name: `nationwide-cities.png`
- Save to: `assets/nationwide-cities.png`
- Size: `1600 x 900`
- Prompt:
`Create a clean collage representing nationwide digital services: skyline hints for Phoenix, Los Angeles, Dallas, Miami, and New York, with overlay UI cards for website, hosting, social media, and print. Modern, minimal, premium, no logos, no readable text, cohesive blue-white color system.`
